SKOS Concept Schemes

Linked Data Vocabularies: Children's Literature Categories

https://doi.org/10.6069/UWLIB.55.B.3

Categories for describing types of historical children's literature RDF vocabulary derived from a thesaurus intended to provide terms for describing categories of historical children's literature. The thesaurus was constructed for use in describing the University of Washington Libraries' Historical Children's Literature Collection which can be seen at https://content.lib.washington.edu/childrensweb/index.html.

Newspaper Genre List

https://doi.org/10.6069/UWLIB.55.D.5

Thesaurus for designating genres of newspapers. A SKOS vocabulary serialized as HTML/RDFa, RDF/XML, nTriples, JSON-LD, and Turtle.

RDF Ontology Extension

RDA Application Profile Extension

https://doi.org/10.6069/UWLIB.55.D.4

Properties required to extend local RDA (Resource Description and Access) application profile. Extension required because RDA did not provide some elements/properties needed to fully describe our resources. Profile was written as part of the LD4P2 grant project in the interest of producing data entry forms in the linked data cataloging environment called Sinopia.

RDF Vocabularies

Linked Data Platform Vocabulary

https://doi.org/10.6069/UWLIB.55.B.1

RDFS vocabulary establishing classes and properties needed to describe platforms used to process RDF datasets. The word platforms represents applications, software, scripts, programs, etc. useful in processing the datasets.

RDF Vocabulary for Linked Data Processes

https://doi.org/10.6069/UWLIB.55.B.2

The RDF triples below define a vocabulary for describing processes run on linked data datasets. This vocabulary was originally intended as administrative and provenance data for RDF datasets produced at the University of Washington Libraries.

Classes: University of Washington Libraries rdfs:Class definitions

https://doi.org/10.6069/UWLIB.55.D.1

This RDF dataset is a description of classes needed to describe University of Washington Libraries resources.

Linked Data Platforms

https://doi.org/10.6069/UWLIB.55.D.2

Platforms (or applications) used to process RDF datasets.

University of Washington Libraries rdf:Property definitions

https://doi.org/10.6069/UWLIB.55.D.3

Describes a digital collection in which an entity is referenced. The digital collection metadata is the source of the data describing the entity. This property is useful for tracking the digital collections that contributed to the local graph.

RDF Dataset Description

University of Washington Libraries' Semantic Web Data

https://doi.org/10.6069/UWLIB.55.A

Landing page for linked data describing resources associated with digital collections hosted at the University of Washington Libraries. The landing page contains a VoID description of the dataset as well as the dataset, which itself is a description of the dataset's partitions. The core data can be found in the partitions.
