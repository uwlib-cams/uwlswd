RDF Dataset Description

University of Washington Libraries' Semantic Web Data

https://doi.org/10.6069/UWLIB.55.A

Landing page for linked data describing resources associated with digital collections hosted at the University of Washington Libraries. The landing page contains a VoID description of the dataset as well as the dataset, which itself is a description of the dataset's partitions. The core data can be found in the partitions.
