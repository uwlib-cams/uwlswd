SKOS Concept Scheme

Linked Data Vocabularies: Children's Literature Categories

https://doi.org/10.6069/uwlib.55.b.3

Categories for describing types of historical children's literature RDF vocabulary derived from a thesaurus intended to provide terms for describing categories of historical children's literature. The thesaurus was constructed for use in describing the University of Washington Libraries' Historical Children's Literature Collection which can be seen at https://content.lib.washington.edu/childrensweb/index.html.

https://doi.org/10.6069/uwlswd.043f-gn39

Special class of motion picture to which the item belongs., e.g., a film reel.

Remote-sensing image: attitude of sensor

General angle of the device from which the remote-sensing image is made.

https://doi.org/10.6069/UWLSWD.06XX-6744

https://doi.org/10.6069/UWLSWD.084W-3778

Sound recording: configuration of playback channels

https://doi.org/10.6069/uwlswd.0av9-3018

Configuration of playback channels for a sound recording based on a clear indication of intended playback. Does not refer to the configuration of channels orginally recorded, unless those channels are all intended to be available on playback.

Tactile material: class of braille writing

https://doi.org/10.6069/uwlswd.0xj7-fz15

Family of braille to which the item belongs. This is not the particular braille code, but the type of braille code used, representing different types of written symbols.

Some: government publication

RDF Vocabulary

Linked Data Platform Vocabulary

https://doi.org/10.6069/uwlib.55.b.1

RDFS vocabulary establishing classes and properties needed to describe platforms used to process RDF datasets. The word platforms represents applications, software, scripts, programs, etc. useful in processing the datasets.

RDF Vocabulary for Linked Data Processes

https://doi.org/10.6069/uwlib.55.b.2

Vocabulary for describing processes run on linked data datasets.

University of Washington Libraries rdfs:Class Definitions

https://doi.org/10.6069/uwlib.55.d.1

RDF dataset describing locally-defined classes used in University of Washington Libraries Semantic Web Data.

Linked Data Platforms

https://doi.org/10.6069/uwlib.55.d.2

Platforms (or applications) used to process RDF datasets.

University of Washington Libraries rdf:Property Definitions

https://doi.org/10.6069/uwlib.55.d.3

RDF dataset describing locally-minted properties used in University of Washington Libraries RDF data.

University of Washington Libraries RDA Application Profile Extension

https://doi.org/10.6069/uwlib.55.d.4

Properties required to extend local RDA (Resource Description and Access) application profile. Extension required because RDA did not provide some elements/properties needed to fully describe our resources. Profile was written as part of the LD4P2 grant project in the interest of producing data entry forms in the Sinopia Linked Data Editor.
